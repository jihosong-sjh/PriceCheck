# Feature Specification: 중고 전자제품 가격 가이드

**Feature Branch**: `001-price-guide`
**Created**: 2025-11-28
**Status**: Draft
**Input**: 팔고 싶은 물품의 사진과 상태를 입력하면 적정 판매가를 추천해주는 중고 물품 가격 가이드 웹 서비스

## User Scenarios & Testing *(mandatory)*

### User Story 1 - 전자제품 가격 추천 받기 (Priority: P1)

사용자가 판매하고 싶은 중고 전자제품의 정보를 입력하고 적정 판매가를 추천받습니다. 제품명, 모델명, 상태를 입력하면 시스템이 시세 데이터를 분석하여 추천 가격과 가격 범위를 제시합니다.

**Why this priority**: 서비스의 핵심 가치를 제공하는 기본 기능으로, 이 기능 없이는 서비스가 성립하지 않습니다. 비회원도 바로 이용할 수 있어 사용자 유입의 핵심 경로입니다.

**Independent Test**: 비회원 사용자가 제품 정보를 입력하고 추천 가격을 확인하는 것만으로 완전한 가치를 제공받을 수 있습니다.

**Acceptance Scenarios**:

1. **Given** 사용자가 가격 추천 페이지에 접속한 상태, **When** 제품명(iPhone 14 Pro), 모델명(128GB), 상태(상)를 입력하고 '가격 추천' 버튼을 클릭, **Then** 추천 가격(예: 850,000원)과 가격 범위(800,000원 ~ 900,000원)가 표시됨
2. **Given** 사용자가 제품 정보를 입력한 상태, **When** 상태를 '중'으로 변경, **Then** 추천 가격이 상태 반영하여 재계산됨
3. **Given** 사용자가 유효하지 않은 제품명을 입력한 상태, **When** '가격 추천' 버튼 클릭, **Then** "해당 제품의 시세 정보를 찾을 수 없습니다" 메시지 표시

---

### User Story 2 - 시세 비교 정보 확인 (Priority: P2)

가격 추천을 받은 사용자가 추천 가격의 근거가 되는 시세 비교 정보를 확인합니다. 유사 매물의 판매 가격, 거래 플랫폼, 등록일 등을 볼 수 있습니다.

**Why this priority**: 추천 가격의 신뢰도를 높이고 사용자가 합리적인 판매 결정을 내릴 수 있도록 지원합니다. P1 기능의 부가 가치를 제공합니다.

**Independent Test**: 가격 추천 결과 화면에서 시세 비교 정보만 확인하는 것으로 테스트 가능합니다.

**Acceptance Scenarios**:

1. **Given** 사용자가 가격 추천을 받은 상태, **When** '시세 비교' 영역 확인, **Then** 최소 3개 이상의 유사 매물 정보(가격, 상태, 플랫폼, 등록일)가 표시됨
2. **Given** 시세 비교 정보가 표시된 상태, **When** 특정 매물 클릭, **Then** 해당 플랫폼의 상세 페이지로 이동(새 탭)

---

### User Story 3 - 사진 업로드 (Priority: P3)

사용자가 제품 사진을 업로드하여 추후 AI 분석 기능 확장에 대비합니다. MVP에서는 사진 저장 및 표시 기능만 제공합니다.

**Why this priority**: 현재는 필수 기능이 아니지만, AI 이미지 분석으로 확장하기 위한 기반을 마련합니다.

**Independent Test**: 사진 업로드 후 정상적으로 표시되는지 확인할 수 있습니다.

**Acceptance Scenarios**:

1. **Given** 가격 추천 페이지에서 제품 정보 입력 중, **When** 사진 업로드 버튼 클릭 후 이미지 파일 선택, **Then** 이미지 미리보기가 표시됨
2. **Given** 사진이 업로드된 상태, **When** 삭제 버튼 클릭, **Then** 업로드된 사진이 제거됨
3. **Given** 지원하지 않는 파일 형식(PDF 등) 선택, **When** 업로드 시도, **Then** "JPG, PNG 형식만 지원합니다" 오류 메시지 표시

---

### User Story 4 - 회원가입 및 추천 히스토리 조회 (Priority: P4)

사용자가 회원가입하여 이전에 받은 가격 추천 기록을 저장하고 조회할 수 있습니다.

**Why this priority**: 부가 기능으로 재방문 사용자에게 편의를 제공합니다. MVP 출시 후 추가해도 됩니다.

**Independent Test**: 회원가입 후 히스토리 페이지에서 이전 추천 기록을 확인할 수 있습니다.

**Acceptance Scenarios**:

1. **Given** 비회원이 가격 추천을 받은 상태, **When** 회원가입 완료, **Then** 방금 받은 추천이 히스토리에 자동 저장됨
2. **Given** 로그인한 회원 상태, **When** 히스토리 페이지 접속, **Then** 이전 추천 기록 목록이 날짜순으로 표시됨
3. **Given** 히스토리 목록 확인 중, **When** 특정 기록 클릭, **Then** 해당 추천의 상세 정보가 표시됨

---

### Edge Cases

- 시세 데이터가 없는 오래되거나 희귀한 제품의 경우 "시세 데이터 부족" 안내 메시지 표시
- 크롤링 데이터 수집 실패 시 "일시적으로 시세 조회가 불가합니다" 메시지와 재시도 옵션 제공
- 동일 제품이지만 상태(상/중/하)에 따른 가격 차이가 클 경우 상태별 가격 범위를 명확히 구분하여 표시
- 사진 업로드 시 파일 크기 제한(10MB) 초과 시 "파일 크기가 너무 큽니다" 안내

## Requirements *(mandatory)*

### Functional Requirements

- **FR-001**: 시스템 MUST 전자제품 카테고리(스마트폰, 노트북, 태블릿, 스마트워치, 이어폰/헤드폰)를 선택할 수 있는 옵션 제공
- **FR-002**: 시스템 MUST 제품명과 모델명을 입력받을 수 있는 폼 제공
- **FR-003**: 시스템 MUST 제품 상태(상/중/하)를 선택할 수 있는 옵션 제공
- **FR-004**: 시스템 MUST 입력된 제품 정보를 기반으로 중고거래 플랫폼의 시세 데이터를 조회
- **FR-005**: 시스템 MUST 시세 데이터를 분석하여 적정 추천 가격과 가격 범위(최저~최고)를 계산
- **FR-006**: 시스템 MUST 추천 가격의 근거가 되는 유사 매물 시세 정보를 표시
- **FR-007**: 시스템 MUST 비회원도 가격 추천 서비스를 이용할 수 있도록 허용
- **FR-008**: 시스템 MUST 사용자가 제품 사진(JPG, PNG)을 업로드할 수 있는 기능 제공
- **FR-009**: 시스템 MUST 회원가입 및 로그인 기능 제공
- **FR-010**: 시스템 MUST 로그인한 회원의 가격 추천 히스토리를 저장 및 조회 기능 제공
- **FR-011**: 시스템 MUST 시세 데이터가 없는 경우 사용자에게 명확한 안내 메시지 표시
- **FR-012**: 시스템 MUST 한국어로 모든 인터페이스 제공

### Key Entities

- **전자제품 (ElectronicItem)**: 사용자가 가격을 알고 싶은 중고 제품. 카테고리, 제품명, 모델명, 상태, 사진(선택)을 속성으로 가짐
- **가격 추천 (PriceRecommendation)**: 시스템이 계산한 추천 판매가. 추천가격, 최저가격, 최고가격, 분석 기준일, 참조한 시세 데이터 목록을 포함
- **시세 데이터 (MarketPrice)**: 중고거래 플랫폼에서 수집한 유사 매물 정보. 플랫폼명, 판매가격, 제품상태, 등록일, 원본 URL을 포함
- **사용자 (User)**: 서비스 회원. 이메일, 비밀번호, 가입일, 추천 히스토리 목록을 가짐
- **추천 히스토리 (RecommendationHistory)**: 회원의 이전 가격 추천 기록. 조회일시, 입력한 제품 정보, 추천 결과를 포함

## Success Criteria *(mandatory)*

### Measurable Outcomes

- **SC-001**: 사용자가 제품 정보 입력부터 가격 추천 결과 확인까지 1분 이내에 완료할 수 있음
- **SC-002**: 가격 추천 결과 표시 시 최소 3개 이상의 유사 매물 시세 정보가 함께 제공됨
- **SC-003**: 비회원 사용자도 제한 없이 가격 추천 서비스를 이용할 수 있음
- **SC-004**: 사용자의 80% 이상이 첫 시도에 가격 추천을 성공적으로 완료함
- **SC-005**: 회원은 최소 30일간의 추천 히스토리를 조회할 수 있음
- **SC-006**: 시스템이 지원하는 전자제품 카테고리에 대해 90% 이상 유효한 시세 데이터 제공

## Assumptions

- 중고거래 플랫폼(번개장터, 중고나라 등)의 데이터를 크롤링할 수 있다고 가정
- 제품 상태는 3단계(상/중/하)로 단순화하여 분류
- MVP에서는 한국 시장만 대상으로 함
- 사진 업로드는 현재 표시 용도로만 사용하며, AI 분석은 추후 확장 예정
- 회원가입은 이메일 기반으로 진행
